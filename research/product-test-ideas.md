# Product test ideas

Dette er testplaner og testideer. Ingen av disse skal omtales som faktisk test før testen er gjennomført og dokumentert.

## Merking

- Faktisk test: Produktet er brukt etter dokumentert metode, med dato, forhold, testere og resultater.
- Research-basert anbefaling: Basert på spesifikasjoner, kilder, tilgjengelighet, kjente bruksområder og diskusjoner.
- Idé til fremtidig test: En test som bør gjennomføres, men som ikke gir grunnlag for konklusjon ennå.

## Produktsider som kan lages uten falsk test

- "Slik velger du putter" kan være guide/research.
- "Beste puttere for nybegynnere" kan publiseres som researchbasert liste hvis alle kort er merket tydelig.
- "Beste disc golf-sekk under 1000 kr" bør helst vente på faktisk test eller publiseres som "aktuelle kandidater".
- "Beste sko til disc golf" bør være testbasert fordi grep og komfort må erfares.
- "Beste kurv til hagebruk" bør ha faktisk putting-/støytest før rangering.
