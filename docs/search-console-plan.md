# Search Console-plan

Denne planen brukes etter lansering av Diskgolfguiden.no.

## Oppsett

1. Legg til `diskgolfguiden.no` i Google Search Console.
2. Velg helst domeneverifisering hvis DNS er tilgjengelig.
3. Bruk URL-prefix hvis domeneverifisering er vanskelig.
4. Send inn `https://diskgolfguiden.no/sitemap.xml`.
5. Sjekk at viktige sider kan indekseres.

## Første kontroll

1. Se etter 404-feil.
2. Se etter sider som er oppdaget, men ikke indeksert.
3. Sjekk mobilbrukervennlighet.
4. Sjekk Core Web Vitals når data finnes.
5. Sjekk at canonical peker riktig.

## Månedlig rutine

1. Sjekk hvilke sider som får visninger.
2. Finn søkefraser med høy impressions og lav CTR.
3. Forbedre title/meta på disse sidene.
4. Finn sider med posisjon 8-20.
5. Legg interne lenker fra relevante huber til disse sidene.
6. Finn søkefraser som fortjener nye artikler.
7. Sjekk 404-feil og fikse interne lenker.
8. Se om banesider, klubbartikler eller produktguider trenger oppdatert kilde.

## Prioritering

- Først: indeksering, feil og sitemap.
- Deretter: CTR på sider med mange visninger.
- Så: internlenking til sider på posisjon 8-20.
- Til slutt: nye artikler basert på faktiske søkefraser.
