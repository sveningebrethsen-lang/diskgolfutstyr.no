# SEO-checklist

## Sideoppsett

- `title` er unik, konkret og under kontroll.
- Meta description beskriver faktisk nytte.
- Én H1 per side.
- H2/H3 følger naturlig struktur.
- Canonical er satt.
- Open Graph og Twitter card er satt på publiserte sider.
- Strukturert data brukes der det gir mening.

## Sitemap og robots

- `sitemap.xml` inneholder publiserte sider.
- Uferdige, interne eller dupliserte sider holdes utenfor sitemap.
- `robots.txt` tillater indeksering og peker til sitemap.

## Internlenker

- Forside lenker til nybegynner, utstyr, regler, baner og tester.
- Guider lenker til relevante neste steg.
- Utstyrssider lenker til tester og sammenligninger.
- Kildelenker har beskrivende tekst.

## Bilder og assets

- Logoer har alt-tekst.
- Dekorative bilder har tom alt og `aria-hidden` når riktig.
- Ikke bruk produktbilder uten rettigheter.

## Innholdskvalitet

- Sider har tydelig målgruppe.
- Påstander som kan endre seg har kilde eller forbehold.
- Oppdatert-dato brukes på artikler og researchbaserte sider.
- Ingen falske ratings, anmeldelser eller testresultater.
