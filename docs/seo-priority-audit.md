# SEO-prioritetsrevisjon for Diskgolfutstyr.no

Oppdatert: 2026-06-18

## Formål

Denne revisjonen prioriterer videre vekst for Diskgolfutstyr.no uten redesign, URL-endringer, backend, database eller massepublisering av tynt innhold.

Målet er å styrke:

- nybegynnerverdi
- norsk organisk trafikk
- lokal SEO
- internlenking
- redaksjonell troverdighet
- praktisk mobilbruk ute på banen

## Kontrollert lokalt

Følgende sider ble kontrollert lokalt for title, meta description, H1, canonical, Open Graph, internlenker og gamle domenereferanser:

| Side | Fil | Status | Kommentar |
|---|---|---|---|
| Forside | `index.html` | OK | Har title, meta, H1, canonical og OG-url med `diskgolfutstyr.no`. Sterk hub, men kan senere få enda tydeligere prioritering av nybegynnerløp. |
| Nybegynnerguide | `nybegynnerguide.html` | OK | God hubstruktur og mange interne lenker. Bør styrkes med mer egen praktisk dybde i støtteartiklene. |
| Regler | `regler.html` | OK | Klar mobilrettet regelinngang. Bør senere utvides med FAQ og konkrete situasjoner. |
| Artikler | `artikler.html` | OK | Fungerer som guidehub. Bør få bedre prioritering etter brukerreise og trafikkpotensial. |
| Tester | `tester.html` | OK | Tydelig skille mellom test/research er viktig. Bør styrkes med metode, teststatus og testkø. |
| Utstyr | `utstyr/index.html` | OK | God kommersiell hub uten å være aggressiv affiliate. Bør få mer prioritert nybegynnerflyt. |
| Baner | `baner/index.html` | OK | Sterk lokal SEO-hub. Bør bygges videre regionvis, ikke masseproduseres. |

Lokalt ble det ikke funnet gamle tracked referanser til `diskgolfguiden.no` eller `Diskgolfguiden`.

## Live-verifisering

Live HTTP/HTTPS kunne ikke verifiseres fra dette miljøet under kontrollen. DNS så riktig ut lokalt, men HTTP/HTTPS-kall returnerte tilkoblingsfeil.

Dette bør kontrolleres manuelt i nettleser og Search Console:

- `https://diskgolfutstyr.no/`
- `https://www.diskgolfutstyr.no/`
- `https://diskgolfutstyr.no/sitemap.xml`
- `https://diskgolfutstyr.no/robots.txt`

## Prioritet 1: Nybegynnerunivers

Dette er viktigste vekstområde. Norske nybegynnere søker etter enkle svar, riktig diskvalg, regler og praktiske kastetips.

| Prioritet | Side/tema | Hvorfor viktig | Anbefalt tiltak |
|---|---|---|---|
| Høy | Beste diskgolfdisk for nybegynnere | Høy søkeintensjon og affiliate-relevant uten å måtte love test. | Styrk med tydelig research-merking, hvem disken passer for, vanlige feilkjøp og internlenker til flight numbers. |
| Høy | Første diskgolfsett | Perfekt startpunkt for nye spillere. | Lag tydelig 3-disk starterbag: putter, midrange, fairway. Ikke press startsett hvis enkeltdisker er bedre. |
| Høy | Driver vs midrange vs putter | Søkes og diskuteres ofte. | Gjør siden ekstremt praktisk med når hver disk brukes, ikke bare definisjoner. |
| Høy | Vanlige nybegynnerfeil | Høy nytte og internlenkingsverdi. | Bygg som feilsøkingsside: hyzer, nose angle, for raske disker, feil grep, dårlig sikting. |
| Høy | Hvordan kaste backhand | Stor trafikkmulighet. | Praktiske steg, vanlige feil, 3 øvelser, video-notater senere. |
| Middels | Hvordan kaste forehand | Viktig for viderekomne nybegynnere. | Skill mellom kontroll-forehand og maks-lengde. |
| Middels | Hvordan putte | Høy nytte, lav kommersiell verdi men sterk autoritet. | Rutine, avstandssoner, enkel treningsplan. |
| Høy | Discgolf regler for nybegynnere | Viktig tillitsside. | Utvid med FAQ og lenker til PDGA som offisiell kilde. |
| Middels | Slik kommer du over 80 meter | God long-tail og praktisk progresjon. | Lag som realistisk utviklingsguide, ikke mirakeloppskrift. |
| Middels | Slik kommer du over 100 meter | Høy interesse, men må være forsiktig. | Forklar teknikk, fysisk kontroll og diskvalg uten raske løfter. |

## Prioritet 2: Baneunivers

Bygg regionvis og kildebasert. Ikke prøv å dekke alle baner med tynne sider.

| Region | Første innholdspakke | Kommentar |
|---|---|---|
| Agder | Regionhub, Kristiansand, nybegynnervennlige baner, klubblenker | Aktuelt for lokal SEO og sommertrafikk. |
| Oslo | Styrk eksisterende Oslo-hub, nybegynnervennlige baner, baner nær Oslo | Høyest trafikkpotensial. |
| Rogaland | Stavanger/Sandnes, klubboversikt, baner å vurdere | Kombiner bysider og regionhub. |
| Vestland | Bergen, lokale baner, klubbstruktur | God lokal SEO-mulighet. |
| Trøndelag | Trondheim, Dragvoll/Trolla/Hallset, klubb/turnering | Sterk discgolfregion og god autoritetsmulighet. |

Alle banesider bør fortsatt bruke forsiktig språk:

- "kan passe for nybegynnere"
- "basert på tilgjengelige kilder"
- "sjekk oppdatert informasjon før du drar"

## Prioritet 3: Utstyrsunivers

Utstyrsklyngen kan gi inntekter senere, men må være tillitsbasert.

| Tema | Prioritet | Merking |
|---|---|---|
| Beste diskgolfdisker for nybegynnere | Høy | Research-basert hvis ikke fysisk testet |
| Beste puttere | Høy | Research/teststatus må vises tydelig |
| Beste midranger | Høy | God nybegynnerverdi |
| Beste kontroll-drivere | Middels | Viktig alternativ til distance drivers |
| Startsett | Høy | Høy kjøpsintensjon |
| Bagger | Middels | Affiliate-potensial, men bør testes fysisk etter hvert |
| Kurver | Middels | Sesong- og gavepotensial |
| Tilbehør | Lav/Middels | Bra støtteinnhold, ikke hovedtrafikk først |

## Prioritet 4: Statiske verktøy

Verktøy kan gi trafikk og lenkeverdi uten backend.

| Verktøy | Verdi | Anbefaling |
|---|---|---|
| Diskvelger | Høy | Statisk JS-basert veiviser med erfaring, kastelengde, kastestil og problem. Resultat bør anbefale disktyper, ikke late som individuell fitting. |
| Kastelengde-guide | Høy | Velg 60, 80, 100, 120+ meter og vis neste tekniske steg. |
| Hvilken disk passer meg? | Høy | Kan kobles til utstyrsguider og flight numbers. Må unngå falsk presisjon. |

## Mobil først

Nåværende struktur har gode quick-paneler og mange direkte innganger. Videre forbedringer bør være små og trygge:

- gjør viktigste handling synlig tidligere på hub-sider
- korte "Hva gjør jeg nå?"-blokker på regler, diskvalg og teknikk
- store trykkflater på kort og CTA-er
- unngå lange tekstblokker før første nyttige lenke
- vurder sticky intern snarvei kun hvis det ikke forstyrrer lesing
- behold høy kontrast og enkle kort

## Autoritet og troverdighet

Viktige autoritetssider finnes, men bør etter hvert få mer presis metode og flere konkrete eksempler:

- `om.html`
- `redaksjonelle-retningslinjer/`
- `docs/physical-test-methodology.md`
- `docs/source-policy.md`
- `docs/affiliate-guidelines.md`
- `affiliate-info.html`

Prioritert forbedring:

1. Forklar tydelig forskjell på fysisk test, research-basert anbefaling og fremtidig test.
2. Vis hvordan kilder brukes.
3. Vis hvordan feil kan meldes inn.
4. Ikke påstå samarbeid eller erfaringer som ikke finnes.

## 30/60/90-dagers plan

### Neste 30 dager

- Fullfør Search Console-verifisering.
- Send inn sitemap.
- Sjekk HTTPS og Page indexing.
- Styrk nybegynneruniverset: diskvalg, første sett, regler og vanlige feil.
- Legg inn flere interne lenker fra forside, nybegynnerhub og utstyrshub.

### Neste 60 dager

- Bygg regionbacklog for Agder, Oslo, Rogaland, Vestland og Trøndelag.
- Forbedre utstyrshubben med tydeligere kjøpsreise.
- Lag første versjon av statisk diskvelger som research-basert veileder.
- Oppdater testmetodikk og tester-hub med konkret testkø.

### Neste 90 dager

- Publiser første regionpakke med høy kvalitet.
- Lag kastelengde-guide for 60/80/100/120+ meter.
- Bruk Search Console-data til å prioritere sider med impressions.
- Forbedre title/meta på sider med høy impressions og lav CTR.
- Begynn å samle faktiske testnotater for produkter som senere kan merkes fysisk testet.

## Anbefalt neste fase

Fase 16B bør være en kontrollert nybegynnerpakke:

- forbedre/utvide 5-7 eksisterende nybegynnersider
- ikke endre URL-er
- ikke lage falske tester
- legge inn bedre internlenking
- oppdatere sitemap kun hvis nye sider faktisk publiseres
