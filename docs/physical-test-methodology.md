# Physical test methodology

Dette dokumentet gjelder fremtidige fysiske tester. Ingen eksisterende research-artikler skal merkes som fysisk test før kravene er oppfylt.

## Krav for fysisk test

En artikkel kan merkes som fysisk test når Diskgolfutstyr faktisk har brukt produktet etter en dokumentert metode.

Minimum:

- produktnavn og variant
- testdato
- teststed
- værforhold
- hvem som testet
- spillerens nivå
- antall kast eller runder
- hva som ble vurdert
- begrensninger

## Disker

For disker bør testen dokumentere:

- minst 20-30 kast per disk hvis mulig
- både rolige og hardere kast
- kast med putter/midrange/fairway der relevant
- vind, temperatur og føre
- banetype eller felt
- hvordan disken oppførte seg for testeren

Sammenlign disker i samme rolle. Ikke sammenlign putter og distance-driver som om de skal løse samme problem.

## Bagger

For bagger bør testen vurdere:

- komfort over hel runde
- kapasitet
- stabilitet når bagen står
- flaskelommer og smårom
- vekt
- regn/vått føre hvis testet
- hva som ikke ble testet

## Kurver

For kurver bør testen vurdere:

- montering
- stabilitet
- lyd/støy
- flytting/lagring
- putting fra korte avstander
- hvor godt den passer hagebruk

## Bilder

Bilder kan brukes når Diskgolfutstyr har tatt dem selv eller har eksplisitt tillatelse. Bilder bør dokumentere produkt, teststed og relevante detaljer.

## Subjektive vurderinger

Subjektive vurderinger skal merkes som subjektive. Eksempel: "For testeren føltes putteren dyp i hånden."
