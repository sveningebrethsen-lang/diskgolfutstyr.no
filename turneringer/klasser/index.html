Klasser i discgolfturneringer

Klasser handler om nivå, alder, kjønn og om du spiller proff eller amatør. Les alltid arrangørens informasjon.

Innhold

  1. Grunnprinsipp
  2. Hvordan velge klasse?

Dette er en statisk, kildebasert side. Informasjon om klubber, turneringer og baner kan endre seg. Sjekk alltid offisiell kilde før du melder deg på, møter opp eller planlegger reise.

Grunnprinsipp

PDGA-arrangementer bruker divisjoner med egne krav. Lokale arrangementer kan ha enklere klasseinndeling. Eventside og arrangør er alltid fasit.

Hvordan velge klasse?

  • Les divisjonskrav.
  • Se om ratinggrense gjelder.
  • Spør arrangør hvis du er ny.
  • Velg lavterskel/amatørklasse når du vil lære formatet.
